## 4. Immediate Skills Alignment
*   **The Migration Lever:** Practical expertise maintaining, decoupling, and decommissioning legacy PHP monoliths, moving relational data safely into modernized or headless stacks without disrupting live production traffic.

*   **The Scale & Concurrency Lever:** Proven record of managing multi-tier web applications handling extreme traffic, preventing database race conditions via deterministic lock-state engines and optimized caching topologies.
*   **The Full-Stack & Secure DevOps Lever:** Comprehensive command of server provisioning, local staging builds, automated deployment packaging, and air-gapped system upgrades under strict security constraints.
