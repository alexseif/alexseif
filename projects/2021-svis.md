# Project Dossier
State-level transactional infrastructure intended to serve the entire vehicle registration and roadworthiness testing lifecycle for the United Arab Emirates. Brought in to execute static code forensics, audit, and chart the path forward for the sovereign technical inspection system expansion.

### Core Architecture & System Engineering
- **Static Code Forensics & Diagnostics:** Deployed onto a heavily compromised, entirely undocumented, and highly fragile legacy framework. Conducted comprehensive static and relational code audits to trace hidden dependencies, decipher custom API contracts, and evaluate database schema integrity under high-volume simulated loads.
- **Strategic Restraint & Recovery Mapping:** Identified a fundamental misalignment between rigid production constraints and optimal engineering execution models. Exercised professional autonomy, stepping away from immediate production, and instead authored a comprehensive structural recovery roadmap outlining technical debt remediations, database normalization paths, and fail-safe API integration protocols.

### Engineering Notes / Edge Cases
Contracted directly by iSoft during a long-term multi-year relationship. This project highlights a core Technical Partner capability: providing high-value diagnostic clarity, code forensics, and defensive engineering strategies for state-level systems.
