# Project Dossier
State-level transactional infrastructure designed to serve the vehicle registration and roadworthiness testing lifecycle for the United Arab Emirates. Engaged to execute static code forensics, audit, and chart the technical inspection system expansion.

### Core Architecture & System Engineering
- **Static Code Forensics & Diagnostics:** Analyzed a legacy framework. Conducted static and relational code audits to trace dependencies, document API contracts, and evaluate database schema integrity under simulated loads.
- **Recovery Mapping:** Identified misalignments between production constraints and engineering execution models. Authored a structural recovery roadmap outlining technical debt remediations, database normalization paths, and API integration protocols.

### Engineering Notes / Edge Cases
Contracted directly by iSoft during a multi-year relationship. This project highlights a core capability: providing diagnostic clarity, code forensics, and defensive engineering strategies for state-level systems.
