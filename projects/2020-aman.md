# Project Dossier
Deployed for the Egyptian Republic under the Ministry of Interior (MOI). Engineered a public safety and vehicle compliance ecosystem handling automated appointment allocation for a national service fleet.

### Core Architecture & System Engineering
- **Multichannel Fleet Orchestration:** Designed and deployed an infrastructure to map, index, and manage resource slots across state facilities, commercial kiosks, and mobile service units.
- **Dynamic Geo-Spatial Allocation:** Engineered scheduling algorithms and relational database schemas to dynamically calculate real-time availability without corrupting state logic during traffic spikes.

### Engineering Notes / Edge Cases
Contracted via iSoft. This project successfully translated physical public assets into operational rules captured in stable web code.

# Use Case Case Study: National Security & Performance Isolation
### 1. Objective & Scope
Digitize and orchestrate the booking pipeline for Technical Inspection Centers, Driving Schools, and Electronic Traffic Units.

### 2. Technical Decisions & Implementation
- **Network Compliance:** Adhered to data governance, encryption protocols, and connection parameters to handle sensitive vehicle identities and citizen profiles securely.
- **Performance Execution:** Optimized backend query execution and map-rendering pipelines to ensure availability during peak operating hours.
